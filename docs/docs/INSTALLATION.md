## Prerequisites

- Node.js version 14.x or later
- npm or yarn

## Installation Steps

Install the package using npm:

```sh
npm install @the-node-forge/jwt-utils
```

Or using yarn:

```sh
yarn add @the-node-forge/jwt-utils
```

For more details, see [API Reference](./API_REFERENCE.md).
